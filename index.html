Search Database

P3 Propane Help Center

In an internship at Boston Environmental, I developed an information help website for clients of the P3 Propane Safety Inspection software. Using a created a scalable tree-based SQL database, the system tracked article interaction and tutorial video watch time. Using collected data the system predicted the user's needs by suggesting popular and related articles in the top section of the website. The website also allowed user login using existing Microsoft Azure User Management and gave appropriate users permission to alter data in the database.

Search Recommendation Implementation

The goal of the database was to make the most efficient help website possible by attempting to predict the user's issue and delivering a suggested article that matched. This was accomplished by implementing a recursive tree search, articles that had higher interaction were suggested to the user. As the user navigated through different topics or narrowed the search via search term, suggested articles changed Dynamically. Related articles recommendations were suggested according to similar users' past interactions.

Disparity Mapping

Disparity Map Estimation

Disparity mapping uses a set of two 2D images taken centimeters apart(called stereo images) to form a depth map of the space. In a research project, I explored the parameters for disparity map creation in situations with unusual subjects and lighting (this process is usually performed by trial and error).I created a program that generated many disparity maps each with a differing value. The image to the left is an output from the program comparing the disparity maps of a cat whose fur creates a challenging scenario for a disparity map. In the disparity map, the brighter areas are closer to the camera and the darker ones are further. Paper GitHub Repo

Image Stitching

Image Stitching

Image stitching is the process of taking two images with overlapping content and combining them into a single larger image. In this project, I researched what differences image overlap has on image homography estimation time. This research found that the best image overlap value was around %50 overlap, this could help reduce the time needed to create larger stitched images. When taking a panorama image, our phones do this many times in a matter of seconds. The process involves involves locating features in an image, matching them, and creating a homography estimation which aligns the images together. Paper GitHub Repo
